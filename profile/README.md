# deariary

**Your life, your diary, fully automatic.**

deariary connects your tools (calendar, chat, code, tasks, and more), collects activity data, and generates a diary entry every day with AI.

- Website: [deariary.com](https://deariary.com)
- App: [app.deariary.com](https://app.deariary.com)
- Public sample diary: [app.deariary.com/diaries/2026/03/07](https://app.deariary.com/diaries/2026/03/07)

## What deariary does

- Collects activity data from connected services
- Generates daily diary entries with multiple writing styles
- Provides weekly/monthly summaries and searchable history
- Supports sharing, export, and account deletion controls
- Handles user data with a privacy-first policy (no model training on user data)

## Integrations

Currently supported: Google Calendar, Todoist, Slack, Toggl Track, Steam, Bluesky, GitHub, Webhook, Weather

## Organization repositories
